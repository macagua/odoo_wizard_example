# Odoo Wizard Example

The **Contacts** add-on customizations for Odoo 13.0.

Added a Wizard window at ``contacts`` app.

## Tested

- Odoo 13 CE version.

## Install

Go to the Apps icons > Apps > searcha the app with the  name addon, e.g. the ``odoo_wizard_example`` and install it.
